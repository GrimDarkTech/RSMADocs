# RSMA
RSMA - набор инструментов, методов и алгоритмов разрабортки систем имитационного моделирования мобильных робототехнических комплексов.
Пакет является расширением для среды разработки Unity.

# Коротоко о проекте
Используя 3D-модели элементов конструкции робота и имитационные модули, релизованные в проекте, разрабатывается виртуальная модель робота.
Модель помещается на виртуальный полигон для тестирования конструкции, электронных компонентов и программы управления.

## Возможности:
- Моделирование механических соединений
- Моделирование электронных компонентов
- Моделирование микроконтроллеров
[Подробнее о возможностях проекта](https://grimdark.ru/)
